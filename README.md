# FUTURE_DS_03 - College Event Feedback Analysis

## Internship Track
Data Science & Analytics

## Task Description
Analyze student feedback from college events using ratings and textual comments to extract insights and recommendations.

## Tools Used
- Google Colab
- Python
- pandas
- seaborn & matplotlib
- TextBlob

## Dataset
Simulated college event feedback dataset with ratings and comments.

## Outcome
- Identified top-rated events
- Performed sentiment analysis on feedback comments
- Visualized satisfaction trends
- Provided recommendations for improving future events
